## install

    cd ~
    git clone https://github.com/guoqiao/vimx.git .vim
    cd .vim
    ./install.sh

That's all!

## basic usage
* tabs: tn to new, tc to close, th to left, tl to right, tm to mv to last
* windows: ctrl + w + w, or ctrl + [hjkl]

## plugin settings
* NERDTree: tt
* NERD-Commenter: leader + c + i
* emmet: tab(.html only)
* autopep8: F8
